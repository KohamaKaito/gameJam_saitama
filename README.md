# TheBartender
　GameJam2021テーマ「わる」から「お酒を割る」を連想し提案。  
お客さんをオーダーを間違わずに提供できるとスコアが加算される。  
お客さんのオーダーを間違えたり提供が遅れるとスコアが引かれる。  
素早く丁寧にカクテルを作り、高得点を目指すゲーム。
